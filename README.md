# Big-Data-Analysis
Project related to analysis of data collected from Twitter, NYT, Common Crawl
#
TOPIC of DATA:

<br/>

        ● Elections <br/>
        ● Government <br/>
        ● Vote <br/>
        ● Political party <br/>
        Used the same set of words to collect data from all 3 data sources: Twitter, New York Times
